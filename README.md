# Project Overview
================

This project is a task management application built using React and CSS for styling. It allows users to create, edit, and delete tasks, as well as categorize them using tags.

## Key Features
---------------

*   Task creation and editing
*   Task deletion
*   Task categorization using tags
*   User-friendly interface

## Tech Stack
------------

*   Frontend: React
*   Styling: CSS
*   JavaScript

## Usage Instructions
-------------------

1.  Create a new task by clicking the "Create Task" button
2.  Edit an existing task by clicking the "Edit" button next to it
3.  Delete a task by clicking the "Delete" button next to it
4.  Categorize a task by clicking the "Add Tag" button and selecting a tag from the dropdown menu
